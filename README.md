# KaspiKassaAPIv3 NativeAPI BPO Driver

## Диагностика ошибки AddIn.None

- Убедиться, что используется клиент 1С x64.
- Проверить экспорты DLL через `dumpbin /EXPORTS`.
- Проверить XML: `type="native"`, `os="Windows"`, `arch="x86_64"`.
- Убедиться, что DLL лежит в корне ZIP и имя файла совпадает с `path` в XML.
- При переустановке удалить старый драйвер из списка и установить пакет заново.
